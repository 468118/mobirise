The Four Pillars of IoE

Behold, the four pillars of the internet of everything (IoE)!

responsive website templates

People

This refers to the connection between people; socially, professinally and for communications.

Data

The data pillar refers to the process of converting data to helpful information and business intelligence.

Internet of Things

This refers to the devices and systems using the internet. Can include any everyday object which utilises the internet.

Process

Procress refers to the actual interaction between each component. It is the process of everything within IoE.

Above is a diagram explaining the four pillars of IoE (internet of everything). As we can see there is interaction between each pillar. To further explain each basic label there is a list below explaining what type of interactions are happening.

Big Data: This term refers to data which is huge in mass and is typically used by businesses and companies to generate analytics. In this sense, big data is being produced by people using the IoE and therefore companies can use this data to allow more options in the devlopment of IoE. It works in a vise-versa pattern and each side (the people and the system's development) benefits.

M2M: This abbreviation stands for Machine to Machine. It refers to the communication between systems and computing nodes. In this instance data is being transfered to and from IoT devices. These devices are also using data for systems such as AI and generated algorithm.

Connection: This refers to the connection and interaction between people (users) and the IoT. This can be interpreted either through the definition of social interaction or scheduled usage of the device.

Personal

Alexa has many different personal features with also a huge array of customisability. Alexa is geared towards personal use at places like homes and has tons of features which complement that aspect. One of the most popular personal features of Alexa is the ability to play music and playlists. Now instead of requiring to use a computer or mobile phone to control music, Alexa uses its voice commands as a way of activation. You can also personalise and create your own playlists and then change the voice commands.

 

Business

Alexa also has features which are complimentry to a buisness enviroment. Alexa can remind you when certain events are near, when deadlines are due and its command creation feature allows for practically any sort of information relay a business would need. Custom command creation allows for people in the office to create a deadline which can then remind you via voice control. Stocks and shares are also an option which can be implemented into Alexa, under a certain threshold it will notfiy you when a price is low/high.

 

Enviromental

Like most modern IoT, Alexa brings benefits to the enviroment and creates more options for effeicent resource usage.
